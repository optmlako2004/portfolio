Analyse de Données Étudiants

Projet de Science des Données

Projet terminé

Projet d'analyse statistique réalisé en Python avec Jupyter Notebook. L'objectif était d'analyser les données d'étudiants (notes, type de bac, sexe) pour en extraire des statistiques descriptives et créer des visualisations pertinentes.

Technologies utilisées :

Python Pandas Matplotlib Jupyter Notebook Excel

Analyses réalisées :

  • Import et lecture de fichiers Excel (.xlsx)
  • Calcul de moyennes (mean) sur les colonnes de notes
  • Calcul d'écarts-types (std) pour mesurer la dispersion
  • Création de diagrammes en bâton (répartition des notes)

Compétences développées :

  • Manipulation de DataFrames avec Pandas
  • Statistiques descriptives
  • Visualisation de données avec Matplotlib
  • Documentation de code en Jupyter

Exemples de Code 

from matplotlib.pyplot import *
import pandas as pd
import matplotlib.pyplot as plt

# Import du fichier Excel
df = pd.read_excel("ExcelEVAL.xlsx")

# Calcul de la moyenne de la Note 5
df['Note 5'].mean()
# Résultat: 9.72

# Calcul de l'écart-type de la Note 8
df['Note 8'].std()
# Résultat: 5.96
